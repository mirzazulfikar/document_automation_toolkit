# Document Automation Toolkit (Google Colab Edition)

A lightweight browser-based PDF automation toolkit designed for HR, operations, and compliance teams.

This project was built to eliminate repetitive document processing tasks without requiring local Python installation or technical setup.

---

## Features

### 1. PDF Merge Automation
Combine multiple PDF files into a single document with one click.

**Useful for:**
- Compliance audits
- Skills assessment reports
- Personality testing results
- Bulk document consolidation

---

### 2. Bulk PDF Rename Automation
Rename multiple PDF files automatically using predefined naming logic.

**Useful for:**
- System migration
- SOP changes
- ISO compliance document restructuring
- Large-scale file reorganization

---

## Why Google Colab?

This toolkit is intentionally designed for Google Colab to minimize setup friction.

Benefits:
- No Python installation required
- No environment variable setup
- No command line usage
- Works directly in browser
- Easy integration with Google Drive

Recommended for:
- HR teams
- Startup operations teams
- Compliance/admin users
- Non-technical users

---

## How to Use

### Step 1
Open the notebook in Google Colab.

### Step 2
Run cells from top to bottom.

### Step 3
Upload your files or connect Google Drive.

### Step 4
Download processed outputs.

## Use Cases

### Compliance Audits
Combine supporting documents into a single audit-ready file.

Rename large file batches according to new naming conventions.

---

## Disclaimer

This tool was built for operational efficiency and convenience.

Always verify outputs manually before using for production or compliance-critical workflows.

Automation improves speed, but human review remains necessary.

---

## Known Limitations

- Bulk renaming logic assumes files already follow a baseline naming convention or contain extractable identifiers
- Best suited for organizations with existing file governance standards or structured naming policies
